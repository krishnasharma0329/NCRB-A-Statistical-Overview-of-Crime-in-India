🚔 NCRB – A Statistical Overview of Crime in India
📌 Project Overview

This project presents a statistical analysis of crime in India using data from the National Crime Records Bureau (NCRB). The analysis focuses on understanding state-wise crime patterns and year-on-year changes to identify trends in criminal activities across the country.

Using Python for data analysis and visualization, the project explores how crime rates vary between states and across different years. The goal is to provide insights that help understand the current crime scenario and its implications for public safety and law enforcement.

🎯 Objectives

Analyze state-wise crime statistics in India

Study year-on-year changes in crime rates

Identify patterns and trends in different crime categories

Provide insights into crime distribution across states

Use Python to perform data analysis and visualization

🛠 Tools & Technologies Used

Python

Pandas – Data cleaning and analysis

NumPy – Numerical operations

Matplotlib / Seaborn – Data visualization

Jupyter Notebook – Analysis environment

📂 Dataset

The dataset used in this project is based on crime reports published by the National Crime Records Bureau (NCRB).

The dataset contains information such as:

State-wise crime statistics

Year-wise crime records

Crime categories and their frequencies

Comparison of crime rates across states

This data helps in identifying crime trends and patterns across India.

📊 Analysis Performed

The following analyses were performed using Python:

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

State-wise crime comparison

Year-wise crime trend analysis

Visualization of crime patterns using charts and graphs

🔍 Key Insights

Crime rates vary significantly across different states.

Some states show consistent increases in certain crime categories.

Year-on-year analysis reveals changing crime patterns over time.

Data visualization helps in understanding crime distribution more clearly.
